# CSS-only games workspace

This folder is reserved for future CSS-only games. Keep game-specific state machines, assets, and route-level modules here so the existing lab pages do not need to be reorganized later.

The shared CSS-only engine lives in `engine/`. It includes board/grid variables, selector-driven state, CSS animation clocks, typed custom properties, actor primitives, controls, and progressive hooks for newer CSS functions.

Implemented game routes:

- `/games/toggle-puzzle`
- `/games/tic-tac-toe`
- `/games/memory-cards`
- `/games/mine-sweeper`
- `/games/snake`
- `/games/icy-tower`
- `/games/doom`
